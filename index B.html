Web Portfolio

RiderBox webpage

The RiderBox webpage's role is to facilitate a theatre performance technical dossier. It should contain skecth of the scenery, light and audio, also it contains all information about a performance.
It will be a personalized webpage with login request. As you register a new account you can add your stuff to it. At the "Home" page appears the list of saved performances, that also is showed at "performances" dropdown button from the header.
It's created to be used from a single account, at "Add new" option you can give the performace's informations separately.

It's not availableon github right now.
